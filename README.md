# Immortal Pongbat #
*A work-in-progress clone of the classic, Mac game, [Mortal Pongbat](https://www.macintoshrepository.org/5394-mortal-pongbat)*

Live version at https://dll.fyi/pongbat

### Left-paddle keys: ###

* **LShift** for up
* **LCtrl** for down
* **Z** for laser

### Right-paddle keys: ###

* **Return** for up
* **RShift** for down
* **/** (slash) for laser
