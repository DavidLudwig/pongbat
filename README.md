# Immortal Pongbat #
*A work-in-progress clone of the classic, Mac game, Mortal Pongbat*

Live version temporarily at https://alum.wpi.edu/~davidl/pongbat/

### Left-paddle keys: ###

* **LShift** for up
* **LCtrl** for down
* **Z** for laser

### Right-paddle keys: ###

* **Return** for up
* **RShift** for down
* **/** (slash) for laser
